# bank_classification_with_NN
W pliku znajduje się notebook przygotowany w Google Colab, który po przygotowaniu modelu opartego na sieciach neuronowych, przyjmuje oraz waliduje dane o kliencie banku od użytkownika, by postarać się odpowiedzieć na pytanie czy klient założy w banku lokatę. Jako źródło posłużyły dane 'bank.csv' pobrane z platformy Kaggle.
